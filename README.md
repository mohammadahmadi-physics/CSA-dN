# Continuous Sensitivity Analysis for δN Formalism
This repository contains the Python code used in the analysis presented in the accompanying paper, arXiv:0000.00000. The code uses Continuous Sensitivity Analysis within the δN formalism to numerically evaluate and plot the power spectrum and the equilateral‑type non‑Gaussianity parameter for an example model, the piecewise linear Starobinsky model. The details of the method are discussed in the paper.
